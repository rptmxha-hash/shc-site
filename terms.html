1. 서비스 개요

본 애플리케이션("SHC 업로드 도구")은 운영자 1인이 자신 소유의 YouTube 채널에 영상을 업로드·관리하기 위해 만든 개인용 자동화 도구입니다. 외부 사용자에게 배포되거나 판매되지 않으며, 운영자 본인만 사용합니다.

2. 이용 범위

본 도구는 운영자 본인의 YouTube 채널에 대해서만 YouTube Data API v3를 통해 영상 업로드, 메타데이터(제목·설명·태그) 설정, 썸네일 설정 기능을 제공합니다. 제3자 채널이나 계정에는 접근하지 않습니다.

3. 책임의 한계

본 도구는 있는 그대로("as-is") 제공되며, 운영자 본인의 개인 프로젝트 용도로만 사용됩니다. 상업적 배포나 제3자 제공을 목적으로 하지 않습니다.

4. Google API 서비스 약관 준수

본 도구는 YouTube API 서비스 약관을 준수합니다.
