1. 이 서비스의 성격

본 애플리케이션("SHC 업로드 도구")은 운영자 1인이 자신 소유의 YouTube 채널에 영상을 업로드·관리하기 위해 만든 개인 자동화 도구입니다. 외부 사용자에게 제공되지 않으며, 운영자 본인 계정 외의 제3자 데이터를 수집·저장·공유하지 않습니다.

2. 접근하는 데이터

본 도구는 Google OAuth 2.0을 통해 운영자 본인의 Google 계정 인증을 받아 YouTube Data API v3(youtube.upload 등)를 사용해 본인 채널에 영상 업로드, 제목·설명·태그·썸네일 설정 등의 작업을 수행합니다. 인증 토큰은 운영자 본인의 로컬 컴퓨터에만 저장되며 외부 서버로 전송되지 않습니다.

3. 데이터 공유

본 도구는 수집·처리하는 어떠한 데이터도 제3자와 공유하지 않습니다.

4. 데이터 보관·삭제

인증 토큰은 운영자가 도구 사용을 중단하거나 직접 삭제할 때까지 로컬에 보관되며, 언제든지 Google 계정 권한 관리 페이지에서 접근 권한을 철회할 수 있습니다.
